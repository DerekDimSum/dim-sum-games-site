# Site assets

Drop image files here using the exact filenames below and the site will pick
them up. Until a file exists, `index.html` shows a styled placeholder in its
place.

## In place ✅
| Filename | What it is | Used for |
|---|---|---|
| `logo-icon.png` | Studio icon mark (512×512, transparent) | Nav, hero, footer |
| `favicon.png` | Downscaled icon (180×180) | Browser tab + Apple touch icon |
