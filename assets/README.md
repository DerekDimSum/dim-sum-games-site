# Site assets

Drop image files here using the exact filenames below and the site will pick
them up. Until a file exists, `index.html` shows a styled placeholder in its
place.

## Studio logo (provided)
| Filename | Which variant | Used for |
|---|---|---|
| `logo-icon.png` | Icon only — the circular sunburst mark, no text (transparent bg preferred) | Favicon, nav, hero |
| `logo-horizontal.png` | Horizontal lockup, black text | Optional: nav / footer wordmark |
| `logo-stacked.png` | Vertical lockup, black text | Optional |

## Social share (optional but recommended)
| Filename | What it is | Size |
|---|---|---|
| `og-image.png` | Link-preview image for LinkedIn/iMessage/etc. | 1200×630 |

Any reasonable format (PNG/SVG/JPG/WebP) and size works — these are just the
names `index.html` references. If yours differ, tell me and I'll rename in the
markup instead.
